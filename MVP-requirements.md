## 1. 产品概述

### 1.1 产品名称
**BgRemover** (暂定)

### 1.2 产品定位
一键式在线图片背景移除工具，面向个人用户和小型电商卖家，提供快速、高质量的抠图服务。

### 1.3 目标用户
| 用户类型 | 使用场景 | 付费意愿 |
|----------|----------|----------|
| 电商卖家 | 商品图白底处理 | 高 |
| 设计师 | 快速抠图素材 | 中 |
| 普通用户 | 证件照/个人照片处理 | 低 |
| 自媒体 | 封面图/配图制作 | 中 |

### 1.4 核心价值主张
- **快**：5 秒内完成处理
- **简单**：上传即得，无需注册
- **高质量**：基于 remove.bg 专业 API
- **隐私**：图片不存储，处理完即删除

## 3. 技术架构

### 3.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端托管 | Cloudflare Pages | 免费、全球 CDN |
| 后端 API | Cloudflare Workers | 无服务器、内存处理 |
| 背景移除 | remove.bg API | 成熟稳定的抠图服务 |
| 域名 | 自定义域名 | 可选，Cloudflare 免费 SSL |

### 3.2 架构图

```
┌─────────────┐
│   用户浏览器  │
└──────┬──────┘
       │ HTTPS
       ▼
┌─────────────────────┐
│  Cloudflare Pages   │  ← 静态资源 (HTML/CSS/JS)
│  (前端界面)          │
└──────┬──────────────┘
       │ /api/remove
       ▼
┌─────────────────────┐
│  Cloudflare Workers │  ← API 代理 (内存流转)
│  (后端逻辑)          │
└──────┬──────────────┘
       │ HTTPS
       ▼
┌─────────────────────┐
│   remove.bg API     │  ← 背景移除服务
└─────────────────────┘
```

### 3.3 数据流

```
1. 用户上传图片 → Cloudflare Pages
2. 前端调用 /api/remove → Cloudflare Workers
3. Workers 转发请求 → remove.bg API
4. remove.bg 返回处理结果 → Workers (内存)
5. Workers 直接返回图片流 → 用户浏览器
6. 图片不落地，无存储
```


## 4. 功能详细设计

### 4.1 首页设计

**页面元素：**
- Logo + 产品名称
- 上传区域（拖拽 + 点击）
- 支持格式提示（JPG/PNG, 最大 25MB）
- 示例图片展示（前后对比）
- 页脚（版权、联系方式）

**上传区域状态：**
| 状态 | 显示内容 |
|------|----------|
| 空闲 | "拖拽图片到此处 或 点击上传" |
| 悬停 | "释放以上传" |
| 上传中 | 进度条 + 百分比 |
| 处理中 | "正在移除背景..." + 加载动画 |
| 完成 | 预览图 + 下载按钮 |
| 错误 | 错误信息 + 重试按钮 |

### 4.2 API 接口设计

**端点：** `POST /api/remove`

**请求：**
```
Content-Type: multipart/form-data

参数:
- image: File (必填)
```

**响应：**
```
成功 (200):
Content-Type: image/png
Content-Disposition: attachment; filename="removed-bg.png"
[图片二进制数据]

失败 (400/500):
Content-Type: application/json
{
  "error": "错误信息"
}
```

**错误码：**
| 状态码 | 说明 |
|--------|------|
| 400 | 图片格式不支持/文件过大 |
| 401 | API Key 无效 |
| 402 | 免费额度已用完 |
| 500 | remove.bg 服务异常 |

### 4.3 前端逻辑

```javascript
// 核心流程
1. 监听文件选择 → 校验格式和大小
2. 显示上传进度 → FormData 发送
3. 调用 /api/remove → 等待响应
4. 接收 Blob → 创建 ObjectURL
5. 显示预览 → 启用下载按钮
6. 错误处理 → 显示友好提示
```

## 7. 风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| remove.bg API 不稳定 | 高 | 添加重试机制，备用 API 方案 |
| 免费额度快速用完 | 中 | 前端显示剩余额度，引导付费 |
| 大图片处理超时 | 中 | Workers 超时设置 30s，前端提示 |
| 恶意刷量 | 中 | IP 限流，Cloudflare 防火墙 |
| 版权纠纷 | 低 | 用户协议声明，不存储图片 |
